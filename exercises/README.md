# GitHub Copilot Exercises

This folder contains hands-on exercises to help you practice using GitHub Copilot. Each exercise is designed to demonstrate different capabilities of Copilot.

## Exercise List

1. **Exercise 1: Basic Functions** (`01-basic-functions.js`)
   - Learn how to generate simple utility functions
   - Practice with string manipulation, math operations

2. **Exercise 2: Working with Data** (`02-data-processing.py`)
   - Data filtering, mapping, and transformation
   - List/array operations

3. **Exercise 3: API Integration** (`03-api-requests.js`)
   - HTTP requests
   - Async/await patterns

4. **Exercise 4: Testing** (`04-testing.js`)
   - Writing unit tests
   - Test cases generation

5. **Exercise 5: Classes and Objects** (`05-classes.py`)
   - Object-oriented programming
   - Class design

## How to Use These Exercises

1. Open each file in VS Code
2. Read the comments/instructions
3. Let Copilot suggest implementations
4. Review and test the suggestions
5. Try variations and experiment!

## Tips

- **Start with comments**: Describe what you want in plain English
- **Be specific**: More detailed comments = better suggestions
- **Iterate**: If you don't like a suggestion, keep typing or try rewording
- **Learn patterns**: Notice how Copilot understands context

Ready to start? Open `01-basic-functions.js` and let's begin!
